# PIC16-Bit Nano Trainer.

## Features.

- 44 Pins PIC16-Bit Family (dsPIC33/PIC24).
- Powered with micro USB, regulated at 3V3.
- Oscillator internal or external MEMS.
- Headers for breadboard and probes.
- I2C with jumpers.
- UART MCP2221A.
- PCB 21x52mm.
