# NHD-C0216CZ.

## Features.

- 2 Lines x 16 Characters.
- ST7032 controller.
- 5x8 dots with cursor.
- Operating voltage : 3V3.
- Backlight : WHITE.
- [Datasheet NHD-C0216CZ-FSW-FBW-3V3.](https://www.newhavendisplay.com/specs/NHD-C0216CZ-FSW-FBW-3V3.pdf)
