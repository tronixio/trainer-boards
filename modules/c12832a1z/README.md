# NHD-C12832A1Z.

## Features.

- 128x32 pixels.
- ST7565R controller.
- Operating voltage : 3V3.
- Backlight : WHITE.
- [Datasheet NHD-C12832A1Z-FSW-FBW-3V3.](https://www.newhavendisplay.com/specs/NHD-C12832A1Z-FSW-FBW-3V3.pdf)
