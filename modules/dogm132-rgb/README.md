# EA-DOGM132-RGB.

## Features.

- 132x32 pixels.
- ST7565R controller.
- SPI.
- Operating voltage : 3V3.
- Backlight : RGB.
- [Datasheet EA DOGM132.](https://www.lcd-module.com/eng/pdf/grafik/dogm132-5e.pdf)
- [Datasheet ST7565R.](https://www.crystalfontz.com/controllers/Sitronix/ST7565R/)
