# NHD-0420DZ - ST7066U.

## Features.

- 4 Lines 20 Characters.
- ST7066U controller (HD44780 Compatible).
- 4-bit, 8-bit.
- Operating voltage : 5V (NHD-0420DZ-FL-YBW).
- Operating voltage : 3V3 (NHD-0420DZ-FL-YBW-33V3).
- Backlight : Yellow/Green.
