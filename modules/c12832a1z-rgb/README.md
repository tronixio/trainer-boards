# NHD-C12832A1Z-RGB.

## Features.

- 128x32 pixels.
- ST7565R controller.
- Operating voltage : 3V3.
- Backlight: RED - GREEN - BLUE.
- [Datasheet NHD-C12832A1Z-FSRGB-FBW-3V3.](https://www.newhavendisplay.com/specs/NHD-C12832A1Z-FSRGB-FBW-3V.pdf)
