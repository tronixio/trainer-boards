# OLED0.96.

## Features.

- 128x64 pixels.
- SSD1306 controller.
- Operating voltage : 3V3 - 5V.
- 3 Colors : BLUE - WHITE - YELLOW/BLUE.
- [Datasheet SSD1306.](https://www.crystalfontz.com/controllers/SolomonSystech/SSD1306/)
